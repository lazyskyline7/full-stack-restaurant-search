
This project uses [Vite](https://vitejs.dev/) for development and build tooling.

## Available Scripts

In the project directory, you can run:


### `yarn start`

Runs the app in development mode using Vite.<br />
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
### `yarn preview`

Previews the production build locally using Vite.<br />
Open [http://localhost:4173](http://localhost:4173) to view it in the browser.



### `yarn test`

Runs the test suite using [Vitest](https://vitest.dev/).


### `yarn build`

Builds the app for production to the `build` folder using Vite.<br />
The build is optimized and ready to be deployed.
